### 1. Clone Repository
Unduh kode sumber dari GitHub dengan perintah berikut:
```bash
git clone https://github.com/username/repository-name.git
cd repository-name
```

### 2. Install Dependencies
```
npm install
```

### 3. 
```
npm run dev
```

### Teknologi yang Digunakan
